# ice-deportation-analysis
Statistical and machine learning analysis of ICE report data to identify factors influencing deportation decisions.

## Collaboration

This project was developed as part of a university assignment in a team of two students.

My contributions included:
- Exploratory data analysis
- Implementation of regression and classification models
- Model evaluation (RMSE, Confusion Matrix, McNemar test)

## Data
The dataset used in this project comes from Kaggle:
**Dataset:** ICE Arrests, Detention and Deportation Data from Kaggle  
**Source:** https://www.kaggle.com/datasets/bwandowando/ice-arrests-detention-and-deportation-data  
**Please download the dataset and place the file "arrests-latest.csv" in the `data/` folder before running the analysis**
